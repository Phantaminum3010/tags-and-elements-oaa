HTML Tags Cheat Sheet

Headings

Headings from `h1` to `h6` are titles of sections or blocks of a page. Every page must have one, and only one, h1 element.

Paragraphs

Paragraphs of text go in `p` tags.

Lists

There are 2 types of lists commonly used in a html document:

  • Unordered lists (`ul`)

    Unordered lists are marked with bullet points and go in the `ul` tags. Each list item goes in a `li` tag.

  • Ordered lists (`ol`)

    Ordered lists are marked with numbers and go in the `ol` tags. Each list item goes in a `li` tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the `href` attribute. Go to DuckDuckGo to search for more information.

Setting `target="_blank"` will open the link in a new window. See MDN for more details.

Anchor tags can also link to other elements within the same page. If we wish to do this, instead of specifying the href attribute's value with the url of some page, we let its value equal to # + the id of the element that we want to move to. For example, this link jumps to the h1 element of our page.

Text emphasis

Italic

The `em` element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browser.

Bold

The `strong` element means you want to emphasize the content between the opening and closing tags. This shows up as bold text in most browser.
